# list
数组 -> 列表
push -> append
pop -> pop
```py
list = [1,2,3]
list.append(4)
list.pop()
```

#### slice
list[start : end : step]
```py
list = [1,2,3]
list[::-1]
```

### del
删除任意一个元素
```py
list = [1,2,3]
del list[1]
```

### + extend()
`+` 连接两个列表，不改变原列表
```py
list = [1,2,3]
list2 = [4,5]
temp = list + list2
print(temp)
```

`extend()` 方法拼接列表
```py
list = [1,2,3]
del list[1]
```
