this的指向

1. 作为普通函数调用，指向全局

2. 作为对象的方法调用，指向该对象

3. new 调用构造器时，构造器总会返回一个对象，通常 构造器里的对象指向返回的这个对象 若构造器显式返回一个 object 类型对象，则此次运算结果最终会返回这个对象，而不是this

4. 作为普通函数调用，指向全局

1. this 在任何情况下都不指向函数的词法作用域

2. this 在运行时绑定，它指向哪里完全取决于函数在哪里执行，只取决于函数的调用方式

这是 this 绑定规则的第一条：默认绑定 独立函数调用时， this指向全局，严格模式下指向 undefined

this 绑定规则的第二条：隐式绑定 调用位置有上下文时或者被某个对象拥有或者包含， this 会绑定到这个上下文对象

this 丢失

getName 方法作为 obj 对象的属性调用时，this 指向obj 对象，因此第一次输出 obj3 的name

而用另外一个变量 getName2 引用 obj3.getName 时，调用 getName2 时，是普通函数调用，所以 this 指向全局。
